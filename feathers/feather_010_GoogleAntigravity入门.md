# Google Antigravity 使用入门

来源：https://codelabs.developers.google.com/getting-started-google-antigravity?hl=zh-cn

## 基本信息

- 时长：约 105 分钟
- 作者：Romin Irani & Mete Atamel
- 最后更新：2026年5月20日
- 条件：个人 Gmail 账号 + Chrome 浏览器
- 平台：Mac / Windows / Linux

## 课程目录

1. **简介** — Antigravity 是智能体式开发平台，IDE 进化为 agent-first 时代
2. **安装** — 本地安装 + 配置
3. **Agent Manager** — 任务控制中心，管理自主智能体
4. **Antigravity 浏览器** — Agent 自带的浏览器，可点击/滚动/输入/截图/DOM捕获
5. **制品 (Artifacts)** — Agent 产出的代码/文件管理
6. **编辑者 (Editor)** — 代码编辑界面
7. **提供反馈意见**
8. **规则和工作流 (Rules & Workflows)** — 自定义 agent 行为
9. **技能 (Skills)** — 扩展 agent 能力
10. **安全与控制** — 权限管理
11. **总结**

## 关键特性

- Agent 可以自主规划、编码、浏览网页
- 浏览器子代理使用专门的模型（与主 agent 不同）
- 支持扩展程序（Chrome Extension）
- 附带免费配额，可用高级模型
- 仅面向个人 Gmail 提供预览版

## 跟你当前使用的关系

你已经在用 Antigravity 作为主力 IDE（agent-first，不喜欢手写代码）。
这个教程可以帮你发现还没用上的功能，特别是：
- Rules & Workflows 自定义 agent 行为
- Skills 扩展
- 浏览器子代理的进阶用法
