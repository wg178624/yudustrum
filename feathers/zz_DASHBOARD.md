## v5.8 版本升级 (2026-05-28)

相对于 v5.7 的改进：

| 维度 | v5.7 | v5.8 |
|------|:----:|:----:|
| 检索管线 | 核心兜+背景兜两层 | 去重→字典序排序→轻度融合 |
| 输出格式 | 灵活模板 | 绝对固定 `【羽渡尘记忆】·实体:内容` |
| 去重 | 无 | 按 entity 去重，同实体只留一条 |
| 融合 | 无 | 同实体多条事实合并为连贯描述 |
| 排序 | 权重优先 | 字典序第一，相关性第二 |
| 缓存命中 | >99% | >99%（维持，字典序对齐更严格） |

测试命令: `python3 /mnt/d/hermes-workspace/scripts/yudustream_v5_cache.py "查询词"`
