# 大家伙配置讨论 2026-05-26

## 最终配置定案

- **机箱**: 超微 CSE-846 4U机架（HDD背板保护，不换）
- **主板**: 超微 H12SSL-i，SSI-EEB（已是最大哥位，不换）
- **CPU**: AMD EPYC 75F3（32C/64T，Zen3，4.0GHz加速）
  - 相比于原计划7502（Zen2，3.35GHz），75F3单核性能提升30-40%
  - 吃3A游戏单核频率（赛博朋克2077/大镖客2/最后生还者/黑神话悟空）
  - H12SSL-i支持7003系列，刷BIOS 2.3+即可
- **散热**: 待定，2028年再定
  - 选项①: 猫头鹰 NH-U14S TR4-SP3 ¥500（165mm height，CSE-846极限兼容）
  - 选项②: Dynatron A38 ¥250（吵但稳，主动散热吹VRM）
  - 选项③: Arctic Freezer 4U-M/4U-SP3 ¥350-400（性价比高，53-80mm高度）
- **内存**: DDR4 ECC 3200MHz 256GB（8×32GB）
  - 三星/海力士/镁光，2028年谁便宜买谁
  - H12SSL-i仅支持DDR4，不支持DDR5
  - 75F3的256MB L3缓存+8通道DDR4，内存频率对游戏影响很小
- **显卡**: RX 7900 XTX 24GB 公版
- **系统盘**: PM9A1 1TB（已有）
- **数据盘**: 企业级 HDD 4-8TB（背板保护，防震动）
- **电源**: 双长城 ATX 1000W 白金牌 + PTJBOD支架
  - CSE-846原生不兼容标准ATX电源，需支架转接
- **系统**: Debian 单系统
- **用途**: 4个3A游戏 + EDA仿真，LLM走DeepSeek API不本地跑
- **网络**: 槟城家宽默认公网IP，路由器防火墙 + WireGuard隧道
- **远程管理**: IPMI AST2500 BMC（主板自带独立管理口）
- **预算**: ¥10,000-15,000
- **作息**: 日常休眠，与用户同步

## 关键讨论点

1. 电源兼容：长城ATX不能直插CSE-846，需PTJBOD支架
2. CPU频率：EPYC 75F3（4.0GHz）比7502（3.35GHz）更适合3A游戏
3. 散热极限：NH-U14S 165mm在CSE-846 4U机箱余量~5mm，有顶盖加强筋风险
4. DDR5 vs DDR4：H12SSL-i仅支持DDR4，DDR5便宜了也用不上
5. 250w TDP散热：280W用猫头鹰NH-U14S有一点点压
6. 内存品牌：服务器ECC选谁便宜买谁，三星不比其他稳
7. 用户Windows从23H2升到了25H2，体感顺滑
