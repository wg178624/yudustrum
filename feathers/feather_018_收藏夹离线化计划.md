# 收藏夹薅下来计划

## 为什么
- 网站会挂、会404、会改版、会被墙
- 收藏夹存的只是链接，不是内容
- 羽毛存的是内容本体 + 标签 + 可检索

## 怎么操作

### 方式一：我帮你爬（推荐）
你把收藏夹里的链接发给我
每条我说"去瞅一眼" → 读内容 → 存羽毛

```bash
# 一次一条
你: "存一下 https://xxx"
我: 抓 → 整理 → 存 feather_NNN
```

### 方式二：批量导出
浏览器导出书签为 HTML 文件
路径：D:\hermes-workspace\bookmarks.html
我批量解析，逐条爬取存羽毛

### 方式三：你手动筛选
打开浏览器收藏夹，扫一遍
把还在意/还没忘的链接挑出来
按一给我

## 建议优先级

T0 — 先薅：
  技术教程/博客
  API 文档
  在线工具页面

T1 — 有空再薅：
  新闻/资讯类（信息有时效性
  论坛帖子（评论区才有价值

不用薅的：
  登录页（Gmail/GitHub/Google
  购物/视频站
  自己的社交账号
