# 教材库学习路线图

## 当前阶段（你在这里）
不需要高数，不需要微积分。需要的是二进制、布尔代数、地址概念。

## 阅读顺序

### T0 — 现在就看
| 教材 | 原因 |
|------|------|
| **计组（唐朔飞）** | 理解内存/地址/指令/CPU → C语言的指针就通了 |
| **C Primer Plus** | 学C语法，配合计组一起看 |
| **数据结构（C语言版）** | 计组+CPrimer之后，练C的实际应用 |

### T1 — 接下来
| 教材 | 原因 |
|------|------|
| **数电（阎石）** | 与或非门→触发器→加法器，计组的硬件基础 |
| **模电（童诗白）** | 运放/反馈/频率响应，控制论的硬件实现 |
| **工程控制论（钱学森）** | 反馈/稳定/鲁棒，学完模电反馈后看 |

### T2 — 以后
| 教材 | 原因 |
|------|------|
| **操作系统** | 进程/内存/文件系统，学完C后看 |
| **计算机网络** | 协议/分层，跟EIE关系不大，不急 |

### T3 — 需要时翻
| 教材 | 原因 |
|------|------|
| **高数（同济七版）** | 模电里遇到微积分/傅里叶时再查 |
| **概率论（浙大四版）** | EDA仿真/信号处理时需要 |
| **C++ Primer** | C学完了再碰，不冲突 |
| **业余无线电** | 兴趣，有空玩 |
| **工程控制论** | 学完模电反馈后读，有惊喜 |

## 一句话路线

```
计组 → C Primer → 数据结构 → 数电 → 模电 → 控制论
                                    ↑
                              T0之后才需要高数
```
