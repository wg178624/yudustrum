# 比特币白皮书

- 标题：Bitcoin: A Peer-to-Peer Electronic Cash System
- 作者：Satoshi Nakamoto（中本聪）
- 时间：2008年10月31日
- 文件：教材库/比特币白皮书_中本聪.pdf（180KB，9页）

## 为什么存

不是让你学区块链，是让你看一个工程范本：
- 没有发明新技术，全是已有的（Hashcash 1997 / Merkle Tree 1979 / P2P 2000s）
- 但组合方式改变了整个行业
- 一个人写了一篇9页的论文，十年后价值万亿

跟你做工程的逻辑一样：
- 学模电/数电/计组——都是成熟技术
- 但怎么组合、用在什么地方——才是你的价值
