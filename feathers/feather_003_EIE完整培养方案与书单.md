# 🪶 第3片羽毛 · EIE 完整培养方案

## 专业概况
- **名称：** 电子信息工程 (Electronic Information Engineering)
- **国家标准：** 教育部高等学校电子信息类专业教学指导委员会制定
- **学制：** 4年
- **学分：** 约160-170学分
- **授予学位：** 工学学士

## 完整四年课程体系

### 📐 第一学年 · 学科基础（约45学分）

#### 数学（4门）
| 课程 | 推荐教材 | 说明 |
|---|---|---|
| **高等数学（上）** | 同济大学《高等数学》第7版 | 极限/导数/积分/微分方程 |
| **高等数学（下）** | 同济大学《高等数学》第7版 | 多元函数/曲线曲面积分/级数 |
| **线性代数** | 同济大学《线性代数》第6版 | 矩阵/向量空间/特征值 |
| **概率论与数理统计** | 浙江大学《概率论与数理统计》第4版或第5版 | 概率/统计/随机过程基础 |

## 自学路线建议（不上大学的话）

```
第一阶段（3-6个月）
  数学：高数上 + 线性代数
  编程：C语言（你已有）
  电路：邱关源《电路》

第二阶段（6-12个月）
  数学：高数下 + 概率论
  电路进阶：模电 + 数电（你已有）
  核心：信号与系统

第三阶段（12-18个月）
  计算机：计组 + OS + 网络（你已有）
  信号处理：DSP + 通信原理
  嵌入式：单片机

第四阶段（18-24个月）
  选修方向 + 项目实践
```

> 电子信息工程的核心逻辑链：
> 数学 → 电路分析 → 模电/数电 → 信号与系统 → 通信/DSP
> 每一步都是下一步的前置，跳不得。
