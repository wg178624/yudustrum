# 使用示例集锦

## 场景：技术问答

当用户问"VPS 配置怎么搞"时，羽渡尘自动注入相关记忆，提供配置细节和注意事项。

## 场景：项目复盘

长期对话中，羽渡尘自动关联之前聊过的决策记录和踩坑经验，避免重复犯同样的错误。

## 场景：学习追踪

系统记录用户的学习进度、已经掌握的知识点，为后续推荐提供上下文。

## 手动调用

```python
# 检索
yudustrum_recall(query="记忆系统架构", top_k=5)

# 查看状态
yudustrum_stats()

# 健康检查
yudustrum_audit()
```
