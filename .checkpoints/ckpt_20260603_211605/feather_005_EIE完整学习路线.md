# 🪶 第3片羽毛 · EIE 完整学习路线

> 合并自：原 003（EIE培养方案）+ 011（教材库路线）+ 012（CSDIY）+ 008（工程控制论）

## 一句话路线

```
计组 → C Primer → 数据结构 → 数电 → 模电 → 控制论
↑                                ↑
T0 开始了                         T1 之后才需要高数
```

## T0 — 现在就看

| 教材 | 原因 |
|------|------|
| **计组（唐朔飞）** | 理解内存/地址/指令/CPU → C 语言的指针就通了 |
| **C Primer Plus 第六版** | 学 C 语法，配合计组一起看，当天概念当天验证 |
| **数据结构（C语言版）** | 计组+C 之后，用 C 写链表/树/图练手 |

配合同步练习：

| 计组章节 | C 练习 |
|----------|--------|
| 二进制/布尔代数 | 写十进制→二进制转换 |
| 内存地址/字节 | `printf("%p")` 打变量地址 |
| 栈帧 | 递归打印每次调用的局部变量地址 |
| Cache 行 | 二维数组行优先 vs 列优先计时 |
| 虚拟内存 | `malloc` 大数组观察地址连续性 |

## T1 — 接下来

| 教材 | 原因 |
|------|------|
| **数电（阎石）** | 与或非门→触发器→加法器，计组的硬件基础 |
| **模电（童诗白）** | 运放/反馈/频率响应，控制论的硬件实现 |
| **工程控制论（钱学森）** | 反馈/稳定/鲁棒。英文原版 *Engineering Cybernetics*（1954 McGraw-Hill）也在库中，可对照读 |

## T2 — 以后

| 教材 | 原因 |
|------|------|
| **操作系统** | 进程/内存/文件系统，学完 C 后看 |
| **计算机网络** | 协议/分层，跟 EIE 关系不大，不急 |

## T3 — 需要时翻

| 教材 | 原因 |
|------|------|
| **高数（同济七版）** | 模电里遇到微积分/傅里叶时再查 |
| **概率论（浙大四版）** | EDA 仿真/信号处理时需要 |
| **C++ Primer** | C 学完了再碰 |
| **业余无线电** | 兴趣 |

## 完整四年课程体系（参考）

| 学年 | 核心课程 |
|:----:|---------|
| 第一年 | 高数上下 + 线代 + 概率论 + C语言 + 电路分析 |
| 第二年 | 复变函数 + 模电 + 数电 + 信号与系统 + 数据结构 + 电磁场 |
| 第三年 | 计组 + OS + 网络 + DSP + 通信原理 + 单片机 |
| 第四年 | 选修（数字IC/嵌入式/AI）+ 毕设 |

**EIE 核心逻辑链：** 数学 → 电路分析 → 模电/数电 → 信号与系统 → 通信/DSP

## 补充资源

- **CSDIY**（csdiy.wiki）：开源计算机自学路线，收录 MIT/Berkeley/Stanford 公开课。你的主线是 EIE 专属路线，CSDIY 适合学完主线后横向拓宽用
- **B站/中国大学MOOC**：几乎所有 EIE 课程都有免费视频
- **MIT OpenCourseWare**：6.002 Circuits and Electronics
- **EDA Playground**：在线 Verilog 仿真，零配置
