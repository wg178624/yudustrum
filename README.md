# 🪶 羽渡尘 · 记忆仓库

## 这里是什么
Hermes 的长期记忆系统。每次聊天的经验、配置、踩坑记录都存这里。

## 目录说明

| 文件夹 | 中文意思 | 里面有什么 |
|--------|---------|-----------|
| feathers/ | 羽毛 | 精炼的知识经验，手动存的 |
| auto-save/ | 自动存盘 | 每次会话结束自动备份的记忆 |
| anchors/ | 识之锚 | 主线进度节点，下次接着聊 |
| compact/ | 太虚剑神 | 上下文压缩后的缓存 |

## 怎么看
直接用 Obsidian 打开这个文件夹就能看所有羽毛文件。
